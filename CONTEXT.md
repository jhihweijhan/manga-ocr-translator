# 漫畫翻譯神器

這個脈絡定義使用者把圖片中的文字轉成目標語言時，專案採用的核心領域語言。

## Language

**翻譯任務**:
使用者上傳單張圖片後，系統擷取圖片中的文字並產出目標語言譯文的一次處理流程。
_Avoid_: 圖片翻譯、漫畫頁處理、翻譯工作

**文字區塊**:
圖片中一個可辨識且可獨立翻譯的文字片段，可能對應到畫面中的一處文字區域。
_Avoid_: OCR 片段、文字框、文字段落

**位置資訊**:
文字區域或文字區塊在圖片中的位置描述；它是輔助對照的資料，不是翻譯任務成立的必要條件。
_Avoid_: 座標、框線、位置

**文字區域**:
圖片中疑似包含文字的位置範圍。
_Avoid_: 偵測框、座標框、裁切區

**文字區域偵測**:
找出圖片中文字區域的過程。
_Avoid_: 版面分析、框選文字、找泡泡

**區塊譯文**:
單一文字區塊對應的目標語言譯文。
_Avoid_: 翻譯文字、譯文、翻譯結果

**OCR 模型**:
翻譯任務中負責從圖片辨識文字區塊的模型。
_Avoid_: 辨識模型、圖片模型、視覺模型

**翻譯模型**:
翻譯任務中負責把文字區塊轉成區塊譯文的模型。
_Avoid_: 文字模型、語言模型、處理模型

**目標語言**:
翻譯任務希望區塊譯文使用的語言，預設為繁體中文。
_Avoid_: 翻譯語言、輸出語言、語言設定

**來源語言提示**:
使用者提供給模型參考的原文語言線索；它不代表模型一定支援或遵守該語言限制。
_Avoid_: 來源語言、輸入語言、原文語言

**提示詞設定**:
翻譯任務使用的 OCR 與翻譯提示詞內容。
_Avoid_: Prompt、系統提示、模型指令

**重新處理**:
使用者在同一個翻譯任務中更換模型或重新執行流程，並以新的結果取代目前的文字區塊與區塊譯文。
_Avoid_: 重跑、再翻譯、更新結果

**修正原文**:
使用者修改文字區塊中的辨識文字。
_Avoid_: 編輯 OCR、改文字、校正文字

**重新翻譯**:
使用者保留目前的文字區塊，只重新產生區塊譯文。
_Avoid_: 再翻譯、重跑翻譯、更新翻譯

**取消處理**:
使用者中止翻譯任務中尚未完成的 OCR 或翻譯階段。
_Avoid_: 停止、取消任務、中斷

**匯出結果**:
使用者將翻譯任務的文字區塊、區塊譯文、模型選擇、語言設定與提示詞設定輸出成檔案。
_Avoid_: 下載結果、存檔、輸出
